# 阶段1 01 章节识别规则抽取

## 目标

先把章节标题识别出来，形成结构化章节列表。

## 输入

1. 已解析出的纯文本

## 输出

建议输出结构：

```json
[
  {
    "title": "第一章 绪论",
    "level": 1,
    "path": ["第一章 绪论"],
    "start_index": 0,
    "end_index": 1200
  }
]
```

## 规则优先级

1. 中文章节编号规则
2. 数字编号规则
3. 英文章节标题规则

## 注意点

1. 不要把普通句子误识别成章节标题
2. `References / 参考文献` 单独保留
3. 标题识别失败时要能回退到普通切分
