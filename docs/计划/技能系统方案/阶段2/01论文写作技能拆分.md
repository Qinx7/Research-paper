# 阶段2：论文写作技能拆分

## 1. 目标

把当前 `PaperWritingAgent` 的职责拆成更清晰的 skill。

## 2. 建议 skill

### 2.1 `paper.chapter_draft`

职责：

- 生成单章节正文草稿

### 2.2 `paper.chapter_grounding`

职责：

- 校验 citations
- 校验真实数据表述
- 应用章节约束

### 2.3 `paper.references_collect`

职责：

- 汇总全文参考文献

### 2.4 `paper.abstract_generate`

职责：

- 生成摘要与关键词

## 3. 当前最小接入建议

第一阶段只真正接入：

1. `paper.chapter_draft`
2. `paper.chapter_grounding`

其他 skill 先定义，不急着全接。
