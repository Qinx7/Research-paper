# 阶段3 01 写作页整篇入口

## 1. 目标

把整篇审查与整篇修订接入 `/writing`，并且仍然保持在同一个页面中完成。

## 2. 页面动作

建议在顶部主操作区新增：

1. `整篇审查`
2. `整篇修订`

建议位置：

- 与 `生成写作计划`
- `生成大纲`
- `生成完整初稿`
- `保存整篇`

保持同一级操作。

## 3. 右侧展示区

新增 3 块：

1. `整篇审查结果`
2. `整篇修订结果`
3. `整篇修订状态`

## 4. 展示要点

### 整篇审查结果

- 总结
- 重点关注区域
- 问题总数
- 高优先级问题

### 整篇修订结果

- 修订摘要
- 已解决问题
- 修订前后片段

### 整篇修订状态

- 已解决数量
- 剩余数量
- 下一步建议

## 5. 本阶段不做

- 章节页跳转弹窗
- 新的 side drawer
- 额外路由
