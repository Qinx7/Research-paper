# 阶段2 01 分析模块映射

## 目标

把 Stitch 的分析模块转换为当前系统真实可用的数据视图，让页面既有仪表盘质感，又不产生虚假结论。

## 模块映射

### 方向评分趋势

- 输入：`filteredDirections`、`previewDirection`、`currentDirection`。
- 展示：每个方向一根柱，柱高来自 `overall` 或 `recommendation_score`。
- 交互：点击柱子切换预览方向。
- 标记：当前方向显示“当前”，预览方向显示高亮边框。

### 研究领域/评分分布

- 输入：`previewDirection.content.scores`、`feasibility_score`、`recommendation_score`。
- 展示：环形评分或进度条，突出综合推荐度。
- 说明：分项展示文献基础、创新性、可行性、数据可得性、论文价值。

### 核心关键词云

- 输入：研究问题、研究方法、创新点、预期产出。
- 展示：按来源不同使用不同视觉权重。
- 空状态：没有内容时显示“暂无关键词，可先生成或刷新研究方向”。

### 领军研究人员替代区

Stitch 的作者模块需要真实作者数据，当前页面无此数据。本轮替换为“创新与风险”：

- 输入：`innovation`、`risks`。
- 展示：创新点列表和风险提醒。
- 动作：风险较高时提示先补充文献依据或调整方向。

### 高影响力论文替代区

当前页面无直接论文列表。本轮替换为“候选方向卡片”：

- 展示方向标题、背景摘要、评分、研究问题。
- 操作：预览、设为当前方向、生成项目设计。
- 已有设计的方向显示“已有设计”标签。

## 验证

- 没有真实论文数据时不出现假论文。
- 没有作者数据时不出现假作者。
- 评分为 0 或缺失时能显示“暂无”，不出现异常高度或 NaN。
- 所有卡片内容可由当前 `PersistedResearchDirection` 推导。
