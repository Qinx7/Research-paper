}

## 3.2 登录/注册页面

新建 `frontend/src/app/login/page.tsx`：
- 登录表单：email + password
- 注册表单：email + username + password
- 登录/注册成功后存储 token → 跳转到首页
- 错误提示

## 3.3 API 拦截

修改 `frontend/src/lib/api.ts`：
- 所有 `fetch` 调用添加 `Authorization: Bearer <token>` 头
- token 从 localStorage 读取
- 401 响应 → 清除 token → 跳转登录页

## 3.4 首页适配

修改 `frontend/src/app/page.tsx`：
- 未登录：显示欢迎页 + 登录/注册入口
- 已登录：显示项目列表（仅当前用户的项目）
- 右上角显示用户名 + 退出按钮
