# 阶段1 01 学术搜索升级

## 目标

让当前学术搜索从“关键词 → 多源召回”升级为“可配置的学术检索任务”。

## 核心改动

### 1. 搜索模式

新增模式字段：

- `quick_search`
- `literature_review`
- `deep_research`

### 2. 搜索过滤

新增可选过滤字段：

- `library_scope`: `all / cn / en`
- `sources`
- `year_from / year_to`
- `min_citation_count`
- `quality_tags`

### 3. 搜索结果结构

搜索结果建议补充：

- `language`
- `quality_flags`
- `relevance_score`
- `freshness_score`
- `final_score`
- `why_selected`

### 4. 检索流程

建议流程：

1. 提取主题词；
2. 拆分中英文关键词；
3. 按模式生成检索策略；
4. 多源召回；
5. 去重；
6. 打分；
7. 返回结果和解释。

## 注意事项

- 不要求首轮就把所有数据源都打通；
- 先在现有 OpenAlex / Semantic Scholar / CNKI / 维普基础上完成结构升级；
- 质量标签先用“可推断标签”，后续再细化。
