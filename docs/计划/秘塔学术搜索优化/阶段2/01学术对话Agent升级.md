# 阶段2 01 学术对话 Agent 升级

## 目标

让 AI 学术对话从“可选检索增强聊天”升级为“面向学术研究问题的检索型 Agent”。

## 核心改动

### 1. 问题分类

建议将用户问题分为：

- 概念解释；
- 文献综述；
- 研究空白；
- 方法比较；
- 选题建议；
- 数据/实验设计；
- 论文写作辅助。

### 2. 检索决策

根据问题类型决定：

- 是否要检索；
- 检索中文、英文还是双语；
- 检索多少篇；
- 是否需要二次扩搜。

### 3. 证据上下文重构

不要只把文献原始字段拼接给模型，建议重构为：

- 证据编号；
- 结论摘要；
- 方法摘要；
- 与当前问题的关联说明；
- 来源信息。

### 4. 回答约束

模型回答时建议强制区分：

- 文献直接支持的结论；
- 基于多文献整合后的推断；
- 当前还缺乏证据的判断。

### 5. 对话体验

建议状态词更细化：

- 正在理解问题；
- 正在规划检索；
- 正在检索中文文献；
- 正在检索英文文献；
- 正在整理证据；
- 正在生成回答。

## 预期收益

- 回答更像研究助手而不是普通聊天模型；
- 引用更可信；
- 综述类问题效果会明显提升。
