# 阶段3 01 项目知识整合

## 目标

把公开学术搜索与项目私有知识整合起来。

## 建议内容

### 1. 可纳入的私有资料

- 项目成果文件；
- 上传文档；
- 开题设计；
- 开题报告；
- 论文草稿；
- 对话中沉淀出的重点结论。

### 2. 检索层区分

建议查询时分成两类语料：

- `external_academic`
- `project_private`

### 3. 回答层区分

回答中分别标记：

- 外部文献证据；
- 项目内资料证据；
- 推断性结论。

### 4. 后续体验方向

- 像“专题”一样维护项目知识空间；
- 支持对某个项目持续追问；
- 支持从对话中回溯引用来源。
