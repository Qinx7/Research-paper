# 阶段 4：导出与答辩衔接

## 目标

完善论文写作完成后的导出和答辩衔接，减少用户在多个页面间找入口。

## 实施内容

1. 导出格式：
   - 支持 DOCX；
   - 如果后端 PDF 可用，支持 PDF；
   - 用统一格式选择控件替代单一下载链接。

2. 答辩衔接：
   - 展示答辩 PPT 大纲页数；
   - 展示是否缺少真实数据；
   - 提供进入项目详情或答辩相关能力入口。

3. 交付状态：
   - 章节完成度；
   - 成果材料数量；
   - 合规检查状态；
   - 导出可用状态。

## 验证方式

- DOCX 下载仍可用。
- PDF 可用时能通过格式选择下载。
- 答辩大纲缺口能清楚展示。
- `npm.cmd run build` 通过。
