# 阶段1 01 标题摘要参考文献规则抽取

## 目标

为 `scholarly_pdf` 增加轻量规则抽取，让结构化预留位真正有值。

## 实施点

1. 增加标题抽取函数
2. 增加摘要抽取函数
3. 增加参考文献区定位函数
4. 在 `extract_text_from_bytes` 的 PDF 结果中附带 `structured_content`

## 推荐输出

```json
{
  "document_kind": "scholarly_pdf",
  "structured_fields": ["title", "abstract", "sections", "references"],
  "structured_content": {
    "title": "...",
    "abstract": "...",
    "references_text": "..."
  },
  "structured_confidence": {
    "title": "low|medium|high",
    "abstract": "low|medium|high",
    "references": "low|medium|high"
  }
}
```

## 验证重点

1. 普通资料 PDF 不应误报完整论文结构
2. 论文型 PDF 的元数据比现在更有信息量
3. 不影响原有 chunk_text 入库流程
