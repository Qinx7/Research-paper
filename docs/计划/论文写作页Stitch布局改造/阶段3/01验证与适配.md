# 阶段3 01 验证与适配

## 目标

完成构建验证和基础体验检查，确保页面改造没有破坏写作流程。

## 验证清单

- `npm.cmd run build` 通过。
- 未登录状态仍显示登录提示。
- 无项目状态仍显示创建/返回提示。
- 有项目但无草稿时可以新建草稿。
- 有草稿时章节能连续编辑。
- 点击章节目录能滚动到对应章节。
- 保存整篇能触发原保存逻辑。
- 项目资料搜索能展示结果并插入当前章节。
- 导出按钮能保留格式选择。

## 适配要求

- 桌面端优先，保持三栏结构。
- 中等宽度下右侧栏可以收窄，但不隐藏关键按钮。
- 小屏暂不做完整移动端设计，至少保证页面可滚动和不重叠。

## 交付说明

完成后说明：

- 改了哪些布局区域。
- 哪些功能被重新映射到右侧 Sources/Assistant。
- 构建是否通过。
- 仍需后续优化的问题。
