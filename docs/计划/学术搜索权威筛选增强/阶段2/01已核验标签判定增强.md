# 阶段2 01 已核验标签判定增强

## 目标

提升当前可以稳定核验的标签识别质量，优先增强 `IEEE / ACM / 北大核心`。

## 原则

1. 宁可保守，不做激进误判
2. 优先利用现有来源元数据
3. 不为提高命中率而牺牲可信度

## 加强方向

### IEEE

可优先利用：

1. 期刊 / 会议名称关键词
2. 出版组织名称
3. DOI / URL / source metadata 中的已知模式

### ACM

可优先利用：

1. 会议名与出版组织关键词
2. ACM Digital Library 相关元信息模式

### 北大核心

可优先利用：

1. 现有白名单期刊名
2. 更稳的别名 / 全称 / 简称匹配
3. 避免模糊命中导致误判

## 本阶段不做

1. 不把 `EI / JCR / 中科院分区` 强行升级为已核验
2. 不引入外部授权目录依赖

## 验证点

1. 当前测试样例中的已核验标签不回退
2. 已核验标签误判率下降
3. 搜索结果卡片中的真实权威标签更稳定
